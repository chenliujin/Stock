换手率也称之周转率（turnover_rate），是指一定时间内市场中股票转手买卖的频率，反应了股票流通性强弱的指标之一。


# 计算公式

换手率 = 成交量 / 流通股本 x 100%

# 换手率区间

0% ~ 1% : 绝对低量

1% ~ 2% : 成交低靡

2% ~ 3% : 成交温和

3% ~ 5% : 成交活跃

5% ~ 8% : 带量

8% ~ 15% : 放量

15% ~ 25% : 巨量

25% ~ 100% : 成交怪异

我们常使用 3% 以下这个标准，并将小于 3% 的成交额称为“无量”，更为严格的标准是 2%
